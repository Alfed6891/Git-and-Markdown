# **Инструкция по работе с git**

## Что такое git

Git - это консольная утилита, для отслеживания и ведения истории изменения файлов, в проекте

## Создание локального репозитория

Для того чтобы создать (инициализировать) локальный репозиторий необходимо в терминале, находясь в папке проекта набрать команду:

    git init

## Проверка состояния репозитория

Чтобы проверить состояние репозитория используют команду:

    git staus

## Добавление файла в раздел проиндексированных файлов

Для перевода файлов из рабочего каталога в раздел проиндексированных файлов используют и последующей фиксации используют команду

    git add
